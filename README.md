# fcc-bar-chart
[A responsive d3 bar chart showing U.S. GDP through 2015](https://github.com/MatthieuPierce/fcc-bar-chart).

## Built with features from:
- d3
    - scale
    - fetch
    - interpolate
    - axis
    - format
- webpack 
    - manually-configured, not scripted
    - Babel
    - HtmlWebpackPlugin
- scss, javascript, html, and the gang
